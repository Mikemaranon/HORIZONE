# HORIZONE Lite Landing

Static commercial landing page ready for GitHub Pages.

## Structure

- `index.html`: main landing page.
- `styles.css`: visual system, layout and responsive design.
- `script.js`: animations, reveal effects, magnetic buttons and interactive mockup.
- `assets/logos/`: HORIZONE brand assets.
- `downloads/`: place release files here.

## Deploy on GitHub Pages

1. Upload this folder to a GitHub repository.
2. Go to **Settings → Pages**.
3. Select the branch and root folder.
4. Replace the files inside `downloads/` with the real HORIZONE Lite builds.

## Download links

Current placeholders:

- `downloads/horizone-lite-windows.zip`
- `downloads/horizone-lite-macos.zip`
- `downloads/horizone-lite-linux.zip`

Update the links in `index.html` if the release filenames change.
